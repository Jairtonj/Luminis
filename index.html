ℹ️ Como funciona o SQLite aqui

O app usa sql.js — o SQLite real compilado para WebAssembly. O banco fica em memória durante o uso e é salvo num arquivo .sqlite no seu celular.

⚠️ Importante: Para não perder dados ao fechar o navegador, salve o banco periodicamente usando "Exportar .sqlite". Na próxima vez, carregue o arquivo salvo.

O arquivo .sqlite pode ser aberto no PC com DB Browser for SQLite (gratuito).
